
# HAR Router System
This project focuses on Human Activity Recognition using Machine Learning.

## Project Overview
This project focuses on recognizing human activities using smartphone sensor data.
The system analyzes motion data and predicts activities such as walking, sitting, and standing.

## Dataset
The dataset used is the UCI Human Activity Recognition (HAR) dataset.
It contains sensor data collected from smartphones for different human activities.

- Total Features: 561
- Activities: Walking, Sitting, Standing, Laying, etc.
- Data collected from multiple subjects

## Techniques Used

- Data Analysis (EDA)
- Feature Scaling (StandardScaler)
- Dimensionality Reduction (PCA)
- Clustering (KMeans)
- Classification (Random Forest)

## Workflow

1. Load dataset
2. Perform EDA (graphs and analysis)
3. Scale features
4. Apply PCA for dimensionality reduction
5. Use KMeans for clustering
6. Train Random Forest model
7. Evaluate using accuracy and confusion matrix

## Results

- Model Accuracy: ~94%
- Good performance across all activity classes
- Confusion matrix shows high correct predictions

## Key Learnings

- Understanding of real-world sensor data
- Importance of feature reduction using PCA
- Application of machine learning models for classification

## How to Run

1. Open the notebook in Google Colab or VS Code
2. Run all cells
3. The dataset will be downloaded automatically
